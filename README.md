Developed a web-based Blood Bank Management System to simplify blood donation and inventory management.
Implemented donor registration and profile management features.
Enabled online appointment booking for blood donations.
Provided real-time monitoring of blood stock availability.
Integrated a blood compatibility checking system for quick donor-patient matching.
Added weather-based donor safety alerts using live API data.
Implemented blood expiry tracking to identify units nearing expiration.
Reduced blood wastage through smart expiry management and prioritization.
Supported hospital blood request handling for emergency situations.
Included a geo-location based donor finder to locate nearby compatible donors.
Developed an admin dashboard for monitoring donors, inventory, and system activities.
Built using React.js, TypeScript, Node.js, Express.js, and Bootstrap.
